MySQL

MySQL8.0-Linux
错误1

Access denied for user ‘root‘@’localhost’ (using password: YES)

密码错误，可能是之前mysql配置没删干净，导致密码生成的log没出现

解决方法
在[mysqld]后添加skip-grant-tables
进入mysql后flush privileges;
然后修改密码ALTER USER ‘root‘@’localhost’ IDENTIFIED BY ‘XXXX’;

错误 2
MySQL安装报错：Couldn‘t open file /etc/pki/rpm-gpg/RPM-GPG-KEY-mysql-2022

需要先运行下面的命令
rpm –import https://repo.mysql.com/RPM-GPG-KEY-mysql-2022

远程连接

添加一个远程用户

CREATE USER ‘remote‘@’%’ IDENTIFIED with mysql_native_password BY ‘XXXX’;

给远程用户增加全部权限

GRANT ALL ON . TO ‘remote‘@’%’;

启动命令
systemctl start mysqld
